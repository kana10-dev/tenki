# 天気アプリ

日本各地の天気予報を表示するシンプルなWebアプリです。

## 機能

- 17都市の天気予報を表示
- 都市名の入力によるオートコンプリート検索
- 天気に応じた背景色の切り替え（晴れ・曇り・雨・雪）
- 手動での天気情報再読み込み

## 対応都市

札幌・仙台・東京・横浜・長野・静岡・名古屋・京都・大阪・神戸・岡山・広島・高松・福岡・熊本・鹿児島・那覇

## 技術スタック

- React 19
- TypeScript
- Tailwind CSS v4
- Vite
- [天気予報 API（tsukumijima）](https://weather.tsukumijima.net/)

## 開発環境のセットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## プレビュー

```bash
npm run preview
```
